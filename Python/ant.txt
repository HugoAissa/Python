TIPOS PRIMITIVOS:

int() = numerico, Exemplo = 7, -4, 0, 59456654456
float() = numeros reais ou de pontos flutuantes, Exemplo = 4.5, 0.7755, -15.223
bool() = valores logicos ou boleanos, Exemplo = True e False
str() = caractere ou string, Exemplo = 'Ola' , '15', ''

Para verficar qual tipo primitivo está sendo usando é print(type(variavel))
